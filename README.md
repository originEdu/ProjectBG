# 배그따라 만들기 전용 프로젝트

## 📂 Directory Structure

- **Content**
  - **Animations**: 캐릭터 및 배경 애니메이션 관련
    - `Aimoffset`: 조준 오프셋
    - `Anims`
    - `BG_Anim`
    - `BlendSpace`: 블렌드 스페이스
  - **Blueprints**: 게임 로직 및 액터 클래스
    - `Character`: 플레이어 및 NPC 로직
    - `Core`: GameMode, PlayerController 등 핵심 시스템
    - `Utils`: math관련 유틸 전역함수
  - **Maps**
